# 参与贡献指南

感谢你对《中论现代性解读》感兴趣！

## 你可以做的几件事

### 1. 报告错误或提建议（最简单）

不需要懂编程，只需要一个 GitHub 账号。

- 点击本页面上方的 **Issues** 标签
- 点击绿色的 **New Issue** 按钮
- 选择 "内容错误报告" 模板，填写你发现的问题
- 提交！

### 2. 提交修改（稍微进阶）

如果你知道如何使用 Git：

1. 点击页面右上角的 **Fork** 按钮，把仓库复制到你的账号下
2. 修改你 Fork 后的文件
3. 提交一个 **Pull Request**（合并请求）

如果你不懂 Git，也可以直接在网页上编辑：

1. 打开你想修改的文件
2. 点击右上角的铅笔图标（Edit this file）
3. 修改内容，填写修改说明
4. 点击 "Propose changes"

### 3. 参与英文翻译

英文翻译是长期项目，任何人都可以参与。请查看 [TRANSLATION.md](TRANSLATION.md)。

## 常见问题

**Q: 我没有哲学背景，可以参与吗？**
A: 当然可以！哪怕是发现一个错别字，或者某句话读起来不通顺，都是很有价值的贡献。

**Q: 我想批评某个哲学观点，可以吗？**
A: 非常欢迎！学术讨论越激烈越好。请在 Issue 中详细阐述你的观点，并引用相关的哲学文献。

**Q: 这本书有商业出版的计划吗？**
A: 目前以开源形式发布。如未来有商业出版计划，将以不同的版本呈现，本开源项目始终免费。
